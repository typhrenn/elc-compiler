# ECL Compiler (Enemy Control Language)

Currently this is merely a shitty prototype, not optimized nor is it done properly, as of now this project exists merely so i can gain atleast some understanding over how to write a pseudo-assembly language, compile it into a self-designed bytecode, and later execute it with a virtual machine.
In the future i plan to rewrite it in a better, more optimal way. The final goal of this project is to make a TOUHOU-like game in an engine i am currently planning to write out of the scraps seen here.
